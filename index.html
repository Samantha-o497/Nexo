📖 Staff Punishment & Ticket Guidelines

Clear, consistent, and documented punishments to keep NEXOVARAMC fair. This page is for staff reference only.

📌 Guidelines
📌 Use these guidelines to keep punishments consistent
🛡️ LifeSteal-focused: protect the heart economy
📝 Always collect evidence before action
💬 Chat Offenses
Spam — Warn → Mute
Disrespect — Warn → Mute
Hate Speech — Instant Ban
⚔️ Hacking Offenses
KillAura / Reach — 30d Ban → Permanent Ban
X-Ray / Exploits — 14d Ban → 30d Ban
AutoClicker / Macros — Warn → 7d Ban → 30d Ban
🎮 Gameplay Offenses
TP Killing / Scamming — Warn → 7d Ban
Bug Abuse — 7d Ban → 30d Ban
Alt Abusing — 30d Ban → Permanent Ban
🚨 Serious Offenses
DDoS / DoX Threats — Permanent Ban
IRL Trading — Permanent Ban
Staff Impersonation — 30d Ban → Permanent Ban
🎫 How to Handle a Ticket
  • Greet the player politely and thank them for opening a ticket
  • Ask for details (IGN, what happened, proof if needed)
  • Punishment appeals - forward to admin+ only
  • Bug reports - collect evidence and send to developers
  • Player reports - request proof, punish if valid
  • Stay professional - no jokes, no bias
  • Close only when solved
  • If unsure, escalate to Sr-Mod/Admin/Manager
⚡ Important: never delete messages in tickets, always keep logs

⚙️ Ticket Control Buttons

These buttons are used by staff when managing tickets:

  • Claim — Assign the ticket to yourself so others know you’re handling it.
  • Lock — Prevents the player from typing, useful when reviewing.
  • Unlock — Reopens chat access for the player in the ticket.
  • Close Ticket — Ends the ticket once it is resolved.
  • Close & Transcript — Closes the ticket and saves a transcript for records.
