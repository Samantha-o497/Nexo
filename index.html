Staff Punishment & Ticket Guidelines

Clear, consistent, and documented punishments to keep NEXOVARAMC fair. This page is for staff reference only.

📌 Use these guidelines to keep punishments consistent
🛡️ LifeSteal-focused: protect the heart economy
📝 Always collect evidence before action

About

NEXOVARAMC is a LifeSteal server: hearts matter. Staff must act quickly on severe offenses (dupes, alts, hacking) and be fair on community matters.

Punishment Guidelines

1️⃣ Chat Offenses

Chat

Spam (3+ msgs/2s) → Warn → 10m Mute → 30m Mute → 1h Mute

Excessive Caps → Warn → 10m Mute → 30m Mute

Toxicity/Disrespect → Warn → 30m Mute → 1h Mute → 1d Mute

Harassment/Targeted Hate → 1h Mute → 1d Mute → 3d Mute → 7d Ban

Advertising (Discord/IP links) → Permanent Ban

Light Self-Promo (YT/Twitch) → Warn → 30m Mute → 1h Mute

2️⃣ LifeSteal Gameplay Offenses

Gameplay

Heart Scamming → 3d Ban → 7d Ban → 14d Ban → Permanent Ban

Alt Abuse → Permanent Ban

TP Trapping → 7d Ban → 14d Ban → Permanent Ban

Scamming Trades → 3d Ban → 7d Ban → 14d Ban → Permanent Ban

Griefing Bases (if disallowed) → Warn → 1d Ban → 7d Ban

Combat Logging → Warn → 1d Ban → 3d Ban → 7d Ban

Teaming in Solos (if forbidden) → Warn → 3d Ban → 7d Ban

3️⃣ Hacking / Cheating

Security

X-Ray / Resource Exploits → 14d Ban → Permanent Ban

KillAura, Reach, AimBot → Permanent Ban

AutoClicker/Macros → 14d Ban → Permanent Ban

Illegal Mods (ESP, FreeCam, InvSee hacks) → Permanent Ban

Duplication (hearts/items) → Permanent Ban

4️⃣ Serious Offenses

Severe

Doxxing / IRL Threats → Permanent Ban

DDOS Threats / Attempts → Permanent Ban + Report to Hosting

Impersonating Staff → 7d Ban → Permanent Ban

Ban / Mute Evasion (alts/VPN) → Extend punishment → Permanent Ban

5️⃣ Staff Permissions by Rank

Roles

Helper / Sr-Helper → Warns, Kicks, Mutes (up to 1h)

Mod / Sr-Mod → Tempbans (up to 7d), handle gameplay issues

Admin / Sr-Admin → Permanent bans, hacker checks, appeals

Manager → Final say in all punishments

⚡ Important: Anything that affects hearts unfairly (dupes, alts, scamming, hacks) = immediate strong punishment. Always collect evidence.

🎫 Ticket Handling (Discord)

When a player opens a ticket on Discord:

  • Greet the player politely and thank them for opening a ticket.
  • Ask for details (IGN, what happened, proof if needed).
  • Punishment appeals → forward to Admin+ only.
  • Bug reports → collect evidence & send to Developers.
  • Player reports → request proof, punish if valid.
  • Stay professional — no jokes, no bias.
  • Close only when solved.
  • If unsure, escalate to Sr-Mod/Admin/Manager.

⚡ Important: Never delete messages in tickets, always keep logs.

Ticket Buttons

✅ Claim
🔒 Lock
🔓 Unlock
🗑️ Close Ticket
🗑️ Close & Transcript
  • ✅ Claim: Take ownership, so other staff know you’re handling it.
  • 🔒 Lock: Stop the player from sending new messages (staff can still reply).
  • 🔓 Unlock: Reopen if you need more info from the player.
  • 🗑️ Close Ticket: Close without saving transcript — use for trivial issues.
  • 🗑️ Close & Transcript: Close and save full logs. Use for reports, punishments, and appeals.

Pro tip: Always collect screenshots and time stamps. If unsure, escalate to Sr-Mod/Admin and document your decision in the ticket.
